Maps on Maps on Maps!

Jul 24th, 2014

The past two weeks I spent a lot of time learning and working with the Google Maps API, well at least part of it. The part I worked with was setting a map on a webpage, setting markers, and mapping directions. To start to understand how to do all these things, I slowly made my way through the doc and code examples, which are incrediably detailed and easy to follow and I highly recommend checking them out. With that said, the doc is also massive and can be a bit overwhelming if you only want to do basic things. On top of that the code examples don’t explain what the Javascript is doing so that can be a bit confusing if you’re a JS noob like I was two weeks ago.

To learn the basics, I did what I usually do when I see new code I don’t understand, I went through it line by line and tried to explain what it was doing in comments. I also then stuck it in CodePen to play around with it. Below are those code samples with my comments expalaining what’s going on and my codepens so you can play around with the code.
